To modify the projects displayed on your portfolio, you'll need to edit the constants.tsx file. This file contains an array called PROJECTS_DATA which holds all the information for each project card.

Here's how you can do it:

  1. Locate the file: Open the constants.tsx file in your project.

  2. Find PROJECTS_DATA: Inside this file, you'll see an array like this:

          export const PROJECTS_DATA: Project[] = [
  {
    id: 'p1',
    title: 'EcoTrack - Sustainability App',
    description: 'A mobile-first web application to help users track their carbon footprint and adopt sustainable habits. Features include progress tracking, community challenges, and educational resources.',
    imageUrl: 'https://picsum.photos/seed/project1/600/400',
    tags: ['React', 'Node.js', 'MongoDB', 'Tailwind CSS', 'PWA'],
    liveUrl: '#',
    repoUrl: '#',
  },
  // ... more projects
];

          export interface Project {
  id: string; // A unique identifier for the project
  title: string; // The title of your project
  description: string; // A brief description

  liveUrl?: string; // Optional: URL to the live demo

  4. Modify the Projects:

    • To add a new project: Copy one of the existing project objects, paste it as a new item in the array (making sure to add a comma , after the preceding object), and then change its id, title, description, imageUrl, tags, and any liveUrl or repoUrl to reflect your new project. Ensure the id is unique.

    • To remove a project: Simply delete the entire object for that project from the PROJECTS_DATA array. Make sure to handle commas correctly if you remove an object from the middle of the array.

    • To edit an existing project: Change the values of the properties (like title, description, imageUrl, etc.) for the project you want to update. For images, you can use a service like https://picsum.photos/seed/your-unique-seed/600/400 for placeholders, or replace it with a direct URL to your project's screenshot.
